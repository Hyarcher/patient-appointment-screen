How to add patients

You can add patients by filling in the input boxes below. The system will reject any fields which are empty. After entering the data, click the 'Add patient' button.

How to delete patients

You can delete patients only when there are patients stored. The drop down menu will be blank and un-clickable when empty. When data has been stored from the add patient button, the names of patients will show and and be able to be selected for deletion.

You can see the display page by clicking the 'See display' button.

To hide this menu, please click on the help logo again.
